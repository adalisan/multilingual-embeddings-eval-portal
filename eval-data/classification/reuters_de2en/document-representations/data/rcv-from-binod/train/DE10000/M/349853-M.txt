Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Montag gegenüber Freitag gesunken . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 29.320 ( 46.399 ) Stück umgesetzt , davon 17.628 ( 32.383 ) Calls und 11.692 ( 14.016 ) Puts . 
die Call-Put-Ratio betrug 1,50 ( 2,31 ) . 
Umsatzspitzenreiter waren wieder Siemens mit 3793 ( 6536 ) Calls und 3901 ( 4498 ) Puts . 
auf den Dax wurden 37.204 ( 89.832 ) Calls und 50.072 ( 78.038 ) Puts gehandelt . 
der Dax-Volatilitätsindex stieg auf 17,78 ( 17,19 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 12.492 ( 24.053 ) Kontrakten mit 3250,5 ( 3263 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 158 ( 152 ) Kontrakten bei 3312 ( 3314 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 41.943 ( 109.336 ) Kontrakte umgesetzt , im März-Bobl-Future 36.533 ( 74.277 ) . 
im Einmonats-Libor-Future gingen über alle Sichten 1353 ( 583 ) Kontrakte um , im Dreimonats-Libor-Future waren es 2160 ( 5231 ) . 
auf den Bund-Future wurden 155 ( 521 ) Call-und 129 ( 767 ) Put-Optionen gehandelt , auf den Bobl-Future 1004 ( 2326 ) Calls und 1950 ( 3410 ) Puts . 
auf den US-Dollar wurden bis 16.00 Uhr 414 ( 328 ) Call-und 568 ( 52 ) Put-Optionen gehandelt .
