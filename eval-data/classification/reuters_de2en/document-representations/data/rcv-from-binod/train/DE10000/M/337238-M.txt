Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der DTB war am Donnerstag wenig verändert . 
davon entfielen 35.868 ( 33.895 ) auf Calls und 13.994 ( 14.937 ) auf Puts . 
die Call-Put-Ratio betrug 2,56 ( 2,26 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 10.109 Calls und 2133 Puts . 
auf den Dax wurden bis 16.00 Uhr 65.012 ( 75.842 ) Calls und 66.147 ( 108.057 ) Puts gehandelt . 
der Dax-Volatilitätsindex fiel auf 15,32 ( 15,49 ) . 
der März-Dax-Future notierte bis dahin nach einem Umsatz von 19.314 ( 18.517 ) Kontrakten mit 3003,5 ( 2995 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 624 ( 627 ) Kontrakten bei 3142,5 ( 3141 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 112.729 ( 96.532 ) Kontrakte umgesetzt , im März-Bobl-Future 97.234 ( 73.352 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 486 ( 910 ) Kontrakte um . 
auf den Bund-Future wurden 1794 ( 1219 ) Call-und 1143 ( 1495 ) Put-Optionen gehandelt , auf den Bobl-Future 1820 ( 200 ) Calls und 2637 ( 5117 ) Puts .
