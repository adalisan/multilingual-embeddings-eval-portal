das Unternehmen teilte am Dienstag in Chicago mit , im zweiten Quartal sei ein Reingewinn von 127,0 ( Vorjahreszeit 37,0 ) Millionen oder 0,56 ( 0,41 ) Dollar je Aktie entstanden . 
die Einnahmen hätten auf 1,031 ( 0,914 ) Milliarden Dollar zugenommen . 
in den ersten sechs Monaten sei bei Einnahmen von 1,977 ( 1,751 ) Milliarden ein Reingewinn von 219,0 ( 108,8 ) Millionen oder 0,96 ( 0,68 ) Dollar erzielt worden .
