Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Montag bis 16.00 Uhr gegenüber dem Vortag etwas gestiegen . 
bis zu diesem Zeitpunkt wurden 24.711 ( 22.023 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 15.881 ( 13.609 ) auf Calls und 8830 ( 8414 ) auf Puts . 
die Call-Put-Ratio stieg auf 1,79 ( 1,61 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 2522 Calls und 1299 Puts . 
auf den Dax wurden 69.008 ( 49.521 ) Calls und 45.899 ( 41.944 ) Puts gehandelt . 
der Dezember-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 11.567 ( 15.821 ) Kontrakten mit 2720 ( 2714 ) Punkten . 
der Dax-Volatilitätsindex fiel auf 10,27 ( 10,43 ) . 
der Dezemberkontrakt des MDax-Future lag nach einem Umsatz von 246 ( 328 ) Kontrakten bei 3031 ( 3022 ) Punkten . 
im Dezember-Bund-Future wurden bis 16.00 Uhr 46.483 ( 51.973 ) Kontrakte umgesetzt , im Dezember-Bobl-Future 48.624 ( 75.222 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 400 ( 1795 ) Call-und 454 ( 1031 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 505 ( 1200 ) Calls und 1100 ( 1520 ) Puts .
