Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Freitag gegenüber dem Vortag gestiegen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 48.802 ( 34.234 ) Stück umgesetzt , davon 32.546 ( 22.443 ) Calls und 16.256 ( 11.791 ) Puts . 
die Call-Put-Ratio betrug 2,00 ( 1,90 ) . 
Umsatzspitzenreiter waren erneut Siemens mit 10.150 ( 5569 ) Calls und 2705 ( 3295 ) Puts . 
auf den Dax wurden 91.766 ( 39.158 ) Calls und 99.962 ( 67.934 ) Puts gehandelt . 
der Dax-Volatilitätsindex stieg auf 15,13 ( 15,03 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 24.644 ( 18.037 ) Kontrakten mit 3172,5 ( 3121,0 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 334 ( 357 ) Kontrakten bei 3270 ( 3259 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 85.152 ( 86.560 ) Kontrakte umgesetzt , im März-Bobl-Future 57.450 ( 88.349 ) . 
im Einmonats-Libor-Future gingen über alle Sichten 670 ( 3228 ) Kontrakte um , im Dreimonats-Libor-Future waren es 3509 ( 12.872 ) . 
auf den Bund-Future wurden 777 ( 875 ) Call-und 2146 ( 2573 ) Put-Optionen gehandelt , auf den Bobl-Future 1350 ( 3280 ) Calls und 6220 ( 5530 ) Puts . 
auf den US-Dollar wurden bis 16.00 Uhr 1165 ( 1325 ) Call-und 1504 ( 347 ) Put-Optionen umgesetzt .
