Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Montag gegenüber Freitag gesunken . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 35.708 ( 41.167 ) Kontrakte umgesetzt , davon 18.788 ( 24.649 ) Calls und 16.920 ( 16.518 ) Puts . 
die Call-Put-Ratio betrug 1,11 ( 1,49 ) . 
Umsatzspitzenreiter waren Siemens mit 2108 Calls und 1701 Puts . 
auf den Dax wurden 27.413 ( 76.054 ) Calls und 43.149 ( 68.228 ) Puts gehandelt . 
der Dax-Volatilitätsindex fiel auf 17,54 ( 17,89 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 13.773 ( 19.153 ) Kontrakten mit 3181,5 ( 3194,5 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 361 ( 364 ) Kontrakten bei 3307 ( 3310 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 41.660 ( 58.677 ) Kontrakte umgesetzt , im März-Bobl-Future 37.534 ( 56.307 ) . 
im Einmonats-Libor-Future gingen über alle Sichten 2087 ( 4367 ) Kontrakte um , im Dreimonats-Libor-Future waren es 3124 ( 8230 ) . 
auf den Bund-Future wurden 476 ( 410 ) Call-und 1510 ( 1273 ) Put-Optionen gehandelt , auf den Bobl-Future 800 ( 2281 ) Calls und 1500 ( 1926 ) Puts . 
auf den US-Dollar wurden bis 16.00 Uhr 948 ( 513 ) Call-und 1158 ( 2486 ) Put-Optionen gehandelt .
