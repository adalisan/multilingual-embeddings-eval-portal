Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) hat am Montag bis 16.00 Uhr gegenüber Freitag zugenommen . 
bis zu diesem Zeitpunkt wurden 17.942 ( 15.089 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 10.982 ( 10.193 ) auf Calls und 6960 ( 4696 ) auf Puts . 
die Call-Put-Ratio fiel auf 1,57 ( 2,08 ) . 
Umsatzspitzenreiter waren erneut Daimler mit 2273 ( 3918 ) Calls und 1456 ( 1106 ) Puts . 
auf den Dax wurden 18.118 ( 15.387 ) Calls und 22.155 ( 21.844 ) Puts gehandelt . 
der Dezember-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 8060 ( 11.000 ) Kontrakten mit 2681,0 ( 2689,0 ) Punkten . 
der Dezember-MDax-Future lag nach einem Umsatz von 2145 ( 251 ) Kontrakten bei 2922,0 ( 2925,0 ) Punkten . 
der Dax-Volatilitätsindex stieg auf 12,24 ( 12,09 ) . 
im Dezember-Bund-Future wurden bis 16.00 Uhr 38.543 ( 62.802 ) Kontrakte umgesetzt , im Dezember-Bobl-Future 61.448 ( 54.140 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 1225 ( 271 ) Call-und 1223 ( 905 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 5770 ( 650 ) Calls und 2050 ( 1917 ) Puts .
