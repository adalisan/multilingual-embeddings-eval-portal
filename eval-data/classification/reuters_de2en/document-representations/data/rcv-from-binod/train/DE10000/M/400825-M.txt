Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Dienstag gestiegen . 
nach Angaben der DTB wurden bis 16.00 Uhr 34.526 ( 21.577 ) Kontrakte umgesetzt , davon 23.644 ( 16.038 ) Calls und 10.882 ( 5539 ) Puts . 
die Call-Put-Ratio fiel auf 2,17 ( 2,89 ) . 
Umsatzspitzenreiter waren Siemens mit 2993 Calls und 1203 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 67.496 ( 85.525 ) Calls und 80.513 ( 70.925 ) Puts . 
der September-Dax-Future notierte um 16 Uhr nach einem Umsatz von 18.357 ( 16.781 ) Kontrakten mit 4047 ( 4012 ) Punkten , der September-MDax-Future nach einem Umsatz von 451 ( 561 ) Kontrakten mit 4055 ( 4056 ) Punkten . 
im September-Bund-Future wurden 108.980 ( 86.305 ) Kontrakte gehandelt , im Bobl-Future 57.864 ( 46.693 ) und im September-Schatz-Future 9197 ( 6971 ) Stück . 
in den Einmonats-Libor-Futures gingen erneut keine Kontrakte um , in den Dreimonats-Libor-Futures 3922 ( 2800 ) Stück . 
auf den Bund-Future wurden 81 ( 596 ) Call-und ein ( 1183 ) Put-Option gehandelt , auf den Bobl-Future 3600 ( 450 ) Calls und 3580 ( 520 ) Puts . 
auf den US-Dollar gingen 1097 ( 102 ) Calls und 50 ( ein ) Puts um .
