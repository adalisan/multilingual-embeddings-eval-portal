Frankfurt ( Reuter ) - An der DTB sind am Montag bis 16.00 Uhr 32.345 ( 38.182 ) Aktienoptionen umgesetzt worden . 
davon entfielen nach Angaben der Börse 21.915 ( 27.290 ) auf Calls und 10.430 ( 10.892 ) auf Puts . 
Umsatzstärkster Wert waren Commerzbank mit 4625 Call-und 677 Put-Optionen . 
auf den Deutschen Aktienindex wurden 47.341 ( 68.264 ) Calls und 61.497 ( 58.993 ) Puts umgesetzt . 
im Dax-Future gingen bis 16.00 Uhr im Septemberkontrakt 20.699 ( 17.763 ) Stück um , wobei der Future zu diesem Zeitpunkt bei 4137 ( 4190 ) Punkten lag . 
der MDax-Future lag bei einem Umsatz von 559 ( 627 ) Kontrakten 3985 ( 4085 ) Punkten . 
der Dax-Volatilitätsindex stellte sich auf 25,56 ( 23,41 ) Zähler . 
im Bund-Future gingen bis 16.00 Uhr 50.625 ( 64.517 ) Kontrakte um , im Bobl-Future 42.927 ( 43.247 ) . 
im Schatz-Future wurden 3865 ( 6412 ) Kontrakte gehandelt . 
Außerdem wurden auf den Bund-Future 432 ( eine ) Call-und 104 ( 582 ) Putoptionen gehandelt . 
beim Bobl-Future waren es 680 ( 320 ) Calls und 2668 ( 2050 ) Puts . 
auf den Einmonatslibor wurden erneut keine Optionen gehandelt , auf den Dreimonatslibor 694 ( 3237 ) . 
auf den US-Dollar wurden 122 ( 679 ) Calls und 207 ( zwölf ) Puts umgesetzt .
