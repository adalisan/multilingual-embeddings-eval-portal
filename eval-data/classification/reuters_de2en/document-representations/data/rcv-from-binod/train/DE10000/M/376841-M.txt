Frankfurt ( Reuter ) - An der DTB hat der Umsatz mit Aktienoptionen am Dienstag zugenommen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 39.728 ( 17.813 ) Kontrakte umgesetzt , davon 25.110 ( 11.483 ) Calls und 14.618 ( 6330 ) Puts . 
die Call-Put-Ratio betrug 1,71 ( 1,81 ) . 
Umsatzspitzenreiter waren Siemens mit 4631 Calls und 3255 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 35.997 ( 14.508 ) Calls und 45.162 ( 14.080 ) Puts . 
der Dax-Volatilitäts-Index gab auf 19,00 ( 19,27 ) nach . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 17.961 ( 8314 ) Kontrakten mit 3439 ( 3377 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 174 ( 129 ) Kontrakten mit 3575 ( 3557 ) Punkten . 
im Juni-Bund-Future wurden 110.522 ( 41.996 ) Kontrakte gehandelt , im Bobl-Future 60.934 ( 29.061 ) und im Juni-Schatz-Future 9695 ( 3167 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 297 ( 399 ) Kontrakte um , im Dreimonats-Libor-Future waren es 2992 ( 1282 ) . 
auf den Bund-Future wurden 544 ( 650 ) Call-und 482 ( 225 ) Put-Optionen gehandelt , auf den Bobl-Future 2126 ( 500 ) Calls und 2976 ( 525 ) Puts . 
auf den US-Dollar gingen 635 ( 451 ) Call-und 590 ( 409 ) Put-Optionen um .
