Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Donnerstag gefallen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 34.275 ( 45.323 ) Kontrakte umgesetzt , davon 24.453 ( 29.340 ) Calls und 10.022 ( 15.983 ) Puts . 
die Call-Put-Ratio betrug 2,41 ( 1,83 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 5087 Calls und 1968 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 44.212 ( 45.213 ) Calls und 37.178 ( 46. 
132 ) Puts . 
der Dax-Volatilitäts-Index fiel auf 18,99 ( 19,13 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 22.424 ( 20.791 ) Kontrakten mit 3588 ( 3608 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 398 ( 401 ) Kontrakten mit 3790 ( 3775 ) Punkten . 
im Juni-Bund-Future wurden 136.584 ( 91.294 ) Kontrakte gehandelt , im Bobl-Future 100.166 ( 96.113 ) und im Juni-Schatz-Future 21.374 ( 14.162 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 450 ( null ) Kontrakte um , im Dreimonats-Libor-Future 4296 ( 3189 ) .

auf den US-Dollar gingen 1074 ( 789 ) Calls und 177 ( 172 ) Puts um .
