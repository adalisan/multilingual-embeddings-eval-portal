Frankfurt ( Reuter ) - An der DTB hat sich der Umsatz mit Aktienoptionen am Freitag gegenüber dem Vortag abgeschwächt . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 35.827 ( Donnerstag 37.219 ) Stück umgesetzt , davon 25.267 ( 25.709 ) Calls und 10.560 ( 11.510 ) Puts . 
die Call-Put-Ratio betrug 2,39 ( 2,23 ) . 
Umsatzspitzenreiter waren Siemens mit 6173 Calls und 2769 Puts . 
auf den Dax wurden 27.424 ( 24.859 ) Calls und 38.352 ( 32.429 ) Puts gehandelt . 
der Dax-Volatilitätsindex fiel auf 16,40 ( 16,65 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 16.923 ( 12.473 ) Kontrakten mit 3045,5 ( 3030,0 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 389 ( 341 ) Kontrakten bei 3190,0 ( 3175,0 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 60.956 ( 53.945 ) Kontrakte umgesetzt , im März-Bobl-Future 48.468 ( 50.561 ) . 
im Einmonats-Libor-Future gingen über alle Sichten 460 ( 600 ) Kontrakte um , im Dreimonats-Libor-Future waren es 2054 ( 2388 ) Kontrakte . 
auf den Bund-Future wurden 3040 ( 290 ) Call-und 2091 ( 316 ) Put-Optionen gehandelt , auf den Bobl-Future 3550 ( 950 ) Calls und 2210 ( 2033 ) Puts . 
auf den US-Dollar wurden bis 16.00 Uhr 818 ( 708 ) Call-und 1343 ( 1663 ) Put-Optionen umgesetzt .
