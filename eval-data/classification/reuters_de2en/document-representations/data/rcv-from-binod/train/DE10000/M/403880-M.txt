Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Mittwoch deutlich gestiegen . 
nach Angaben der DTB wurden bis 16.00 Uhr 40.683 ( Dienstag 27.386 ) Kontrakte umgesetzt , davon 28.238 ( 17.850 ) Calls und 12.445 ( 9536 ) Puts . 
die Call-Put-Ratio stieg auf 2,26 ( 1,87 ) . 
der Umsatz mit Dax-Optionen verteilte sich auf 96.648 ( 59.928 ) Calls und 95.692 ( 64.807 ) Puts . 
der September-Dax-Future notierte um 16 Uhr nach einem Umsatz von 20.612 ( 18.534 ) Kontrakten mit 4240,0 ( 4177,0 ) Punkten , der September-MDax-Future nach einem Umsatz von 788 ( 414 ) Kontrakten mit 4159 ( 4134 ) Punkten . 
im September-Bund-Future wurden 90.256 ( 153.984 ) Kontrakte gehandelt , im Bobl-Future 65.058 ( 105.069 ) und im September-Schatz-Future 11.054 ( 22.375 ) Stück . 
in den Einmonats-Libor-Futures gingen keine ( 640 ) Kontrakte um , in den Dreimonats-Libor-Futures 3829 ( 5251 ) Stück . 
auf den Bund-Future wurden 716 ( 955 ) Call-und 217 ( 670 ) Put-Optionen gehandelt , auf den Bobl-Future 1650 ( 3220 ) Calls und 11.824 ( 8835 ) Puts . 
auf den Dollar gingen 45 ( 514 ) Calls und neun ( sieben ) Puts um .
