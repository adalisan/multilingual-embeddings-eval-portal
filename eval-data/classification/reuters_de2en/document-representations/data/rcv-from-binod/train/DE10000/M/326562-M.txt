Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Montag bis 16.00 Uhr im Tagesvergleich geschrumpft . 
bis zu diesem Zeitpunkt wurden 17.011 ( 21.153 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 8319 ( 11.205 ) auf Calls und 8692 ( 9948 ) auf Puts . 
die Call-Put-Ratio fiel auf 0,95 ( 1,12 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 2892 Calls und 1863 Puts . 
auf den Dax wurden bis 16 Uhr 17.012 ( 22.441 ) Calls und 23.182 ( 26.938 ) Puts gehandelt . 
der September-Dax-Future notierte nach einem Umsatz von 6797 ( 10.979 ) Kontrakten bei 2560,0 ( 2564,0 ) Punkten . 
der Dax-Volatilitätsindex stieg auf 11,24 ( 11,00 ) . 
im September-Bund-Future wurden bis 16 Uhr 25.475 ( 63.903 ) Kontrakte umgesetzt , im Bobl-Future 20.798 ( 120.020 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 69 ( 726 ) Call-und 630 ( 364 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 454 ( 3777 ) Calls und 200 ( 1895 ) Puts .
