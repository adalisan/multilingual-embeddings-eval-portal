Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Montag bis 16.00 Uhr gesunken . 
bis zu diesem Zeitpunkt wurden 16.392 ( Freitag 22.198 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 10.196 ( 15.287 ) auf Calls und 6196 ( 6911 ) auf Puts . 
die Call-Put-Ratio fiel auf 1,64 ( 2,21 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 1955 Calls und 361 Puts . 
auf den Dax wurden 25.230 ( 32.110 ) Calls und 24.512 ( 49.576 ) Puts gehandelt . 
der September-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 5299 ( 11.087 ) Kontrakten bei 2544,0 ( 2540,0 ) Punkten . 
der Dax-Volatilitätsindex stieg auf 12,36 ( 11,66 ) . 
im September-Bund-Future wurden bis 16.00 Uhr 28.160 ( 31.272 ) Kontrakte umgesetzt , im Bobl-Future 46.525 ( 55.938 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 236 ( 84 ) Call-und 149 ( 839 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 2200 ( 850 ) Calls und 1500 ( 102 ) Puts .
