Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Montag gefallen . 
davon entfielen 21.916 ( 27.347 ) auf Calls und 8994 ( 10.812 ) auf Puts . 
die Call-Put-Ratio betrug 2,43 ( 2,52 ) . 
Umsatzspitzenreiter waren erneut Deutsche Bank mit 5638 ( 6886 ) Calls und 2187 ( 1215 ) Puts . 
auf den Dax wurden 31.553 ( 51.039 ) Calls und 26.080 ( 69.089 ) Puts gehandelt . 
der Dax-Volatilitätsindex stieg auf 16,66 ( 16,13 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 11.968 ( 22.990 ) Kontrakten mit 3002,0 ( 3009,5 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 522 ( 150 ) Kontrakten bei 3176,0 ( 3179,0 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 47.621 ( 91.479 ) Kontrakte umgesetzt , im März-Bobl-Future 29.611 ( 92.191 ) .
