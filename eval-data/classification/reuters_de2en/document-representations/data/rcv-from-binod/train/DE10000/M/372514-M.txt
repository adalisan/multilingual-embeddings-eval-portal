Frankfurt ( Reuter ) - An der DTB hat der Umsatz mit Aktienoptionen am Donnerstag zugenommen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 40.055 ( 33.700 ) Kontrakte umgesetzt , davon 28.563 ( 22.449 ) Calls und 11.492 ( 11.251 ) Puts . 
die Call-Put-Ratio betrug 2,48 ( 1,99 ) . 
Umsatzspitzenreiter waren erneut Daimler mit 4286 ( 5180 ) Calls und 2877 ( 2403 ) Puts . 
auf den Dax wurden 64.215 ( 47.376 ) Calls und 47.001 ( 53.190 ) Puts gehandelt . 
der Dax-Volatilitätsindex stieg auf 21,49 ( 20,93 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 19.926 ( 16.779 ) Kontrakten mit 3373 ( 3348 ) , der Juni-MDax-Future nach einem Umsatz von 345 ( 319 ) Kontrakten mit 3570 ( 3576 ) Punkten . 
im Juni-Bund-Future wurden 115.786 ( 112.563 ) Kontrakte gehandelt , im Bobl-Future 87.589 ( 57.071 ) und im Juni-Schatz-Future 15.874 ( 6457 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten erneut keine Kontrakte um , im Dreimonats-Libor-Future waren es 4463 ( 3216 ) . 
auf den Bund-Future wurden 1046 ( 673 ) Call-und 632 ( 1385 ) Put-Optionen gehandelt , auf den Bobl-Future 279 ( 4245 ) Calls und 3821 ( 6614 ) Puts . 
auf den US-Dollar gingen 476 ( 878 ) Call-und 36 ( 156 ) Put-Optionen um .
