Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Mittwoch gegenüber dem Vortag gestiegen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 48.034 ( 34.429 ) Stück umgesetzt , davon 34.452 ( 21.129 ) Calls und 13.582 ( 13.300 ) Puts . 
die Call-Put-Ratio betrug 2,53 ( 1,58 ) . 
Umsatzspitzenreiter waren erneut Siemens mit 10.968 ( 5196 ) Calls und 2611 ( 3427 ) Puts . 
auf den Dax wurden 70.931 ( 90.686 ) Calls und 88.374 ( 59.140 ) Puts gehandelt . 
der Dax-Volatilitätsindex stieg auf 16,20 ( 15,84 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 29.966 ( 16.267 ) Kontrakten mit 3220,0 ( 3204,5 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 464 ( 288 ) Kontrakten bei 3314 ( 3295 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 76.738 ( 74.269 ) Kontrakte umgesetzt , im März-Bobl-Future 72.231 ( 66.278 ) . 
im Einmonats-Libor-Future gingen über alle Sichten 1340 ( 1742 ) Kontrakte um , im Dreimonats-Libor-Future waren es 5070 ( 4059 ) . 
auf den Bund-Future wurden 597 ( 448 ) Call-und 2047 ( 1370 ) Put-Optionen gehandelt , auf den Bobl-Future 3920 ( 3960 ) Calls und 4102 ( 4228 ) Puts . 
auf den US-Dollar wurden bis 16.00 Uhr 2028 ( 2721 ) Call-und 462 ( 1402 ) Put-Optionen umgesetzt .
