Wien ( Reuter ) - Die Oesterreichische Nationalbank ( OeNB ) läßt die Leitzinsen unverändert . 
die OeNB teilte am Donnerstag in Wien weiter mit , sie biete auch den nächsten Mengentender zu einem unveränderten Zins von 3,00 ( 3,00 ) Prozent an . 
Diskont-und Lombardsatz waren von der OeNB zuletzt per 19. April 1996 auf 2,50 ( 3,00 ) Prozent bzw. 4,75 ( 5,25 ) Prozent zurückgenommen worden . 
der Zinssatz für Offenmarktgeschäfte ( GOMEX ) wurde per 2. September 1996 auf 3,40 ( 3,50 ) Prozent gesenkt .
