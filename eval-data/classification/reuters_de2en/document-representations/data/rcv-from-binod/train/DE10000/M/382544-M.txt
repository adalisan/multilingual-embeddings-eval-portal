Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Dienstag gefallen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 33.397 ( Freitag 54.379 ) Kontrakte umgesetzt , davon 18.693 ( 38.052 ) Calls und 14.704 ( 16.327 ) Puts . 
die Call-Put-Ratio betrug 1,27 ( 2,33 ) . 
Umsatzspitzenreiter waren erneut Deutsche Bank mit 3564 ( 8218 ) Calls und 2222 ( 4043 ) Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 33.041 ( 91.164 ) Calls und 47.658 ( 69.072 ) Puts . 
der Dax-Volatilitäts-Index stieg auf 19,56 ( 18,04 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 18.874 ( 15.646 ) Kontrakten mit 3522 ( 3599 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 508 ( 491 ) Kontrakten mit 3721 ( 3760 ) Punkten . 
im Juni-Bund-Future wurden 77.443 ( 75.039 ) Kontrakte gehandelt , im Bobl-Future 62.529 ( 56.525 ) und im Juni-Schatz-Future 6500 ( 8594 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 1410 ( null ) Kontrakte um , im Dreimonats-Libor-Future 2650 ( 4328 ) . 
auf den Bund-Future wurden 1175 ( 551 ) Call-und 4636 ( 844 ) Put-Optionen gehandelt , auf den Bobl-Future 2060 ( 4871 ) Calls und 7750 ( 652 ) Puts . 
auf den US-Dollar gingen 236 ( 190 ) Calls und 125 ( 578 ) Puts um .
