Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Montag gesunken . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 25.540 ( 32.139 ) Kontrakte umgesetzt , davon 17.255 ( 20.556 ) Calls und 8285 ( 11.583 ) Puts . 
die Call-Put-Ratio betrug 2,08 ( 1,77 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 3064 Calls und 1551 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 49.294 ( 72.228 ) Calls und 86.393 ( 69.373 ) Puts . 
der Dax-Volatilitäts-Index stieg auf 19,71 ( 19,06 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 19.887 ( 17.652 ) Kontrakten mit 3671 ( 3686 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 663 ( 778 ) Kontrakten mit 3906 ( 3886 ) Punkten . 
im September-Bund-Future wurden 87.725 ( 80.447 ) Kontrakte gehandelt , im Bobl-Future 66.975 ( 60.064 ) und im September-Schatz-Future 9199 ( 6660 ) Stück . 
in den Einmonats-Libor-Futures gingen 420 ( 108 ) Kontrakte um , in den Dreimonats-Libor-Futures 5806 ( 1255 ) Stück . 
auf den Bund-Future wurden 106 ( 542 ) Call-und 780 ( 343 ) Put-Optionen gehandelt , auf den Bobl-Future 3525 ( 3855 ) Calls und 3572 ( 2502 ) Puts . 
auf den US-Dollar gingen 761 ( 957 ) Calls und 845 ( 651 ) Puts um .
