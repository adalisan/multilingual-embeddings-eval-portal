Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Freitag gegenüber Donnerstag gesunken . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 23.828 ( 38.958 ) Kontrakte umgesetzt , davon 14.205 ( 26.709 ) Calls und 9623 ( 12.249 ) Puts . 
die Call-Put-Ratio betrug 1,47 ( 2,18 ) . 
Umsatzspitzenreiter waren Thyssen mit 3306 ( 8273 ) Calls und 1718 ( 2761 ) Puts . 
auf den Dax wurden 34.794 ( 53.783 ) Calls und 35.953 ( 45.664 ) Puts gehandelt . 
der Dax-Volatilitätsindex stieg auf 17,88 ( 17,59 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 17.037 ( 20.854 ) Kontrakten mit 3271,5 ( 3277,0 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 289 ( 413 ) Kontrakten bei 3362,0 ( 3351,5 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 115.622 ( 108.737 ) Kontrakte umgesetzt , im März-Bobl-Future 76.072 ( 85.362 ) . 
im Einmonats-Libor-Future gingen über alle Sichten 1503 ( 1575 ) Kontrakte um , im Dreimonats-Libor-Future waren es 5278 ( 4913 ) . 
auf den Bund-Future wurden 767 ( 224 ) Call-und 308 ( 1515 ) Put-Optionen gehandelt , auf den Bobl-Future 2720 ( 1295 ) Calls und 950 ( 1403 ) Puts . 
auf den US-Dollar wurden bis 16.00 Uhr 457 ( 314 ) Call-und 192 ( null ) Put-Optionen gehandelt .
