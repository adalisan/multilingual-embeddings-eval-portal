Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Mittwoch gestiegen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 45.323 ( 33.397 ) Kontrakte umgesetzt , davon 29.340 ( 18.693 ) Calls und 15.983 ( 14.704 ) Puts . 
die Call-Put-Ratio betrug 1,83 ( 1,27 ) . 
Umsatzspitzenreiter waren Daimler mit 4825 Calls und 4539 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 45.213 ( 33.041 ) Calls und 46.132 ( 47.658 ) Puts . 
der Dax-Volatilitäts-Index fiel auf 19,13 ( 19,56 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 20.791 ( 18.874 ) Kontrakten mit 3608 ( 3522 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 401 ( 508 ) Kontrakten mit 3775 ( 3721 ) Punkten . 
im Juni-Bund-Future wurden 91.294 ( 77.443 ) Kontrakte gehandelt , im Bobl-Future 96.113 ( 62.529 ) und im Juni-Schatz-Future 14.162 ( 6500 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten null ( 1410 ) Kontrakte um , im Dreimonats-Libor-Future 3189 ( 2650 ) . 
auf den Bund-Future wurden 727 ( 1175 ) Call-und 601 ( 4636 ) Put-Optionen gehandelt , auf den Bobl-Future 2630 ( 2060 ) Calls und 7652 ( 7750 ) Puts . 
auf den US-Dollar gingen 789 ( 236 ) Calls und 172 ( 125 ) Puts um .
