Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Montag gefallen . 
nach Angaben der DTB wurden bis 16.00 Uhr 30.753 ( Freitag 31.810 ) Kontrakte umgesetzt , davon 21.309 ( 22.497 ) Calls und 9444 ( 9313 ) Puts . 
die Call-Put-Ratio sank leicht auf 2,25 ( 2,41 ) . 
Umsatzspitzenreiter waren erneut Deutsche Bank mit 3589 ( 4385 ) Calls und 1433 ( 2671 ) Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 96.780 ( 44.055 ) Calls und 86.050 ( 45.245 ) Puts . 
der September-Dax-Future notierte um 16 Uhr nach einem Umsatz von 15.847 ( 14.444 ) Kontrakten mit 4164,5 ( 4067,0 ) Punkten , der September-MDax-Future nach einem Umsatz von 441 ( 344 ) Kontrakten mit 4134 ( 4086 ) Punkten . 
im September-Bund-Future wurden 119.552 ( 103.629 ) Kontrakte gehandelt , im Bobl-Future 87.976 ( 53.791 ) und im September-Schatz-Future 18.074 ( 10.752 ) Stück . 
in den Einmonats-Libor-Futures gingen 1955 ( 100 ) Kontrakte um , in den Dreimonats-Libor-Futures 6314 ( 2302 ) Stück . 
auf den Bund-Future wurden 599 ( 72 ) Call-und 2230 ( 500 ) Put-Optionen gehandelt , auf den Bobl-Future 2880 ( 4115 ) Calls und 2330 ( 570 ) ( Puts . 
auf den Dollar gingen 1708 ( 221 ) Calls und 38 ( 185 ) Puts um .
