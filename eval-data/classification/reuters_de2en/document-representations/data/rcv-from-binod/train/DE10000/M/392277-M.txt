Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Mittwoch gesunken . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 26.098 ( 28.732 ) Kontrakte umgesetzt , davon 19.112 ( 18.669 ) Calls und 6986 ( 10.063 ) Puts . 
die Call-Put-Ratio betrug 2,73 ( 1,85 ) . 
Umsatzspitzenreiter waren Siemens mit 5044 Calls und 653 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 49.985 ( 37.668 ) Calls und 32.144 ( 48.825 ) Puts . 
der Dax-Volatilitäts-Index stieg auf 20,66 ( 20,65 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 11.946 ( 17.748 ) Kontrakten mit 3678 ( 3670 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 450 ( 279 ) Kontrakten mit 3887 ( 3872,5 ) Punkten . 
im September-Bund-Future wurden 81.276 ( 64.442 ) Kontrakte gehandelt , im Bobl-Future 49.495 ( 48.072 ) und im September-Schatz-Future 7375 ( 5541 ) Stück . 
in den Einmonats-Libor-Futures gingen 386 ( 971 ) Kontrakte um , in den Dreimonats-Libor-Futures 8749 ( 5522 ) Stück . 
auf den Bund-Future wurden 105 ( 200 ) Call-und 301 ( 1110 ) Put-Optionen gehandelt , auf den Bobl-Future 2575 ( 3800 ) Calls und 617 ( 1030 ) Puts . 
auf den US-Dollar gingen 410 ( 263 ) Calls und 326 ( 1075 ) Puts um .
