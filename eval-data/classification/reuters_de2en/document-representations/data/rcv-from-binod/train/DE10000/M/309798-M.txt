Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Freitag bis 16.00 Uhr gegenüber dem Vortag gestiegen . 
bis zu diesem Zeitpunkt wurden 41.183 ( 39.822 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 26.630 ( 25.822 ) auf Calls und 14.553 ( 14.000 ) auf Puts . 
die Call-Put-Ratio fiel auf 1,82 ( 1,84 ) . 
Umsatzspitzenreiter waren erneut Daimler mit 6308 ( 5301 ) Calls und 2991 ( 640 ) Puts . 
auf den Dax wurden 106.142 ( 39.635 ) Calls und 74.022 ( 38.710 ) Puts gehandelt . 
der Dezember-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 16.926 ( 11.493 ) Kontrakten mit 2737 ( 2733 ) Punkten . 
der Dezember-MDax-Future lag nach einem Umsatz von 378 ( 239 ) Kontrakten bei 3010 ( 3013 ) Punkten . 
der Dax-Volatilitätsindex fiel auf 10,49 ( 10,84 ) . 
im Dezember-Bund-Future wurden bis 16.00 Uhr 55.159 ( 78.172 ) Kontrakte umgesetzt , im Dezember-Bobl-Future 51.734 ( 58.101 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 600 ( 87 ) Call-und 716 ( 105 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 2140 ( 1370 ) Calls und 1830 ( 920 ) Puts .
