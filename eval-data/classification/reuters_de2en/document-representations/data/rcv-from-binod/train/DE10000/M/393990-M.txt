Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Dienstag gestiegen . 
nach Angaben der DTB wurden bis 16.00 Uhr 25.157 ( Montag 21.944 ) Kontrakte umgesetzt , davon 17.772 ( 13.789 ) Calls und 7385 ( 8155 ) Puts . 
die Call-Put-Ratio stieg auf 2,40 ( 1,69 ) . 
Umsatzspitzenreiter waren erneut Deutsche Bank mit 2928 ( 1868 ) Calls und 927 ( 1404 ) Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 53.209 ( 44.209 ) Calls und 65.967 ( 44.586 ) Puts . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 34.913 ( 14.609 ) Kontrakten mit 3715,5 ( 3753,5 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 2485 ( 884 ) Kontrakten mit 3902,0 ( 3925,0 ) Punkten . 
im September-Bund-Future wurden 129.699 ( 78.047 ) Kontrakte gehandelt , im Bobl-Future 96.684 ( 67.545 ) und im September-Schatz-Future 12.654 ( 10.132 ) Stück . 
in den Einmonats-Libor-Futures gingen 200 ( keine ) Kontrakte um , in den Dreimonats-Libor-Futures 2989 ( 1227 ) Stück . 
auf den Bund-Future wurden 222 ( 140 ) Call-und 1102 ( 222 ) Put-Optionen gehandelt , auf den Bobl-Future 1620 ( 800 ) Calls und 1950 ( 1201 ) Puts . 
auf den US-Dollar gingen 670 ( 393 ) Calls und 590 ( 176 ) Puts um .
