Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Donnerstag bis 16.00 Uhr gegenüber dem Vortag geschrumpft . 
bis zu diesem Zeitpunkt wurden 20.802 ( 26.616 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 11.737 ( 15.611 ) auf Calls und 9065 ( 11.005 ) auf Puts . 
die Call-Put-Ratio fiel auf 1,29 ( 1,41 ) . 
Umsatzspitzenreiter waren Daimler mit 1891 Calls und 1252 Puts . 
auf den Dax wurden 33.609 ( 27.017 ) Calls und 46.214 ( 31.474 ) Puts gehandelt . 
der Dezember-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 15.367 ( 14.556 ) Kontrakten mit 2680,5 ( 2666,5 ) Punkten . 
der Dezember-MDax-Future lag nach einem Umsatz von 319 ( 793 ) Kontrakten bei 2911,0 ( 2931,0 ) Punkten . 
der Dax-Volatilitätsindex stieg auf 12,53 ( 11,86 ) . 
im Dezember-Bund-Future wurden bis 16.00 Uhr 46.412 ( 69.411 ) Kontrakte umgesetzt , im Dezember-Bobl-Future 58.085 ( 96.064 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 41 ( 127 ) Call-und 60 ( 396 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 2120 ( 1097 ) Calls und 600 ( 993 ) Puts .
