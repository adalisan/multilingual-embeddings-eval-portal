Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Mittwoch gegenüber Dienstag weiter gesunken . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 28.465 ( 34.551 ) Kontrakte umgesetzt , davon 18.468 ( 23.375 ) Calls und 9997 ( 11.176 ) Puts . 
die Call-Put-Ratio betrug 1,84 ( 2,09 ) . 
Umsatzspitzenreiter waren erneut Thyssen mit 3683 ( 7130 ) Calls und 1646 ( 1389 ) Puts . 
auf den Dax wurden 29.422 ( 70.917 ) Calls und 49.305 ( 55.843 ) Puts gehandelt . 
der Dax-Volatilitätsindex stieg auf 17,38 ( 16,74 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 15.750 ( 19.832 ) Kontrakten mit 3260,5 ( 3241,0 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 261 ( 479 ) Kontrakten bei 3346,5 ( 3336 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 44.761 ( 54.081 ) Kontrakte umgesetzt , im März-Bobl-Future 70.607 ( 76.354 ) . 
im Einmonats-Libor-Future gingen über alle Sichten 2102 ( 2400 ) Kontrakte um , im Dreimonats-Libor-Future waren es 2200 ( 5049 ) . 
auf den Bund-Future wurden sieben ( keine ) Call-und 1219 ( 609 ) Put-Optionen gehandelt , auf den Bobl-Future 1160 ( 776 ) Calls und 2050 ( 1562 ) Puts . 
auf den US-Dollar wurden bis 16.00 Uhr 574 ( 970 ) Call-und 528 ( 182 ) Put-Optionen gehandelt .
