Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Freitag gesunken . 
nach Angaben der DTB wurden bis 16.00 Uhr 23.353 ( Donnerstag 29.448 ) Kontrakte umgesetzt , davon 14.536 ( 18.922 ) Calls und 8817 ( 10.526 ) Puts . 
die Call-Put-Ratio fiel auf 1,64 ( 1,79 ) . 
Umsatzspitzenreiter waren Telekom mit 2585 Calls und 1643 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 27.060 ( 39.705 ) Calls und 37.551 ( 69.451 ) Puts . 
der September-Dax-Future notierte um 16 Uhr nach einem Umsatz von 13.750 ( 14.414 ) Kontrakten mit 3836,0 ( 3832,5 ) Punkten , der September-MDax-Future nach einem Umsatz von 230 ( 195 ) Kontrakten mit 3980 ( 3982 ) Punkten . 
im September-Bund-Future wurden 88.143 ( 117.924 ) Kontrakte gehandelt , im Bobl-Future 51.525 ( 58.039 ) und im September-Schatz-Future 6813 ( 10.321 ) Stück . 
in den Einmonats-Libor-Futures gingen 100 ( 400 ) Kontrakte um , in den Dreimonats-Libor-Futures 1104 ( 2713 ) Stück . 
auf den Bund-Future wurden 493 ( 768 ) Call-und 320 ( 535 ) Put-Optionen gehandelt , auf den Bobl-Future 2150 ( 450 ) Calls und 1100 ( 3620 ) Puts . 
auf den US-Dollar gingen 1047 ( 34 ) Calls und 1691 ( 254 ) Puts um .
