Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Mittwoch bis 16 Uhr im Tagesvergleich gefallen . 
bis zu diesem Zeitpunkt wurden 22.556 ( Dienstag : 25.182 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 11.357 ( 14.831 ) auf Calls und 11.199 ( 10.351 ) auf Puts . 
die Call-Put-Ratio fiel auf 1,01 ( 1,43 ) . 
Umsatzspitzenreiter waren Thyssen mit 1763 Calls und 2274 Puts . 
auf den Dax wurden bis 16 Uhr 30.048 ( 22.116 ) Calls und 44.467 ( 32.959 ) Puts gehandelt . 
der September-Dax-Future notierte nach einem Umsatz von 15.956 ( 10.221 ) Kontrakten bei 2541 ( 2569 ) Punkten . 
der Dax-Volatilitätsindex stieg auf 11,34 ( 11,13 ) . 
im September-Bund-Future wurden bis 16 Uhr 58.675 ( 26.807 ) Kontrakte umgesetzt , im Bobl-Future 119.868 ( 51.726 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 395 ( 361 ) Call-und 872 ( 168 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 4866 ( 950 ) Calls und 5529 ( 227 ) Puts .
