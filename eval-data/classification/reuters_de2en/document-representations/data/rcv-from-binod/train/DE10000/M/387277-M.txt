Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Dienstag weiter geschrumpft . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 25.801 ( 27.436 ) Kontrakte umgesetzt , davon 17.735 ( 16.692 ) Calls und 8066 ( 10.744 ) Puts . 
die Call-Put-Ratio betrug 2,19 ( 1,55 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 1785 Calls und 1677 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 37.893 ( 50.067 ) Calls und 30.459 ( 40.672 ) Puts . 
der Dax-Volatilitäts-Index fiel auf 20,01 ( 20,80 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 20.346 ( 21.444 ) Kontrakten mit 3637 ( 3611,5 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 215 ( 462 ) Kontrakten mit 3812 ( 3796 ) Punkten . 
im Juni-Bund-Future wurden 118.780 ( 84.946 ) Kontrakte gehandelt , im Bobl-Future 99.257 ( 66.078 ) und im Juni-Schatz-Future 19.379 ( 13.217 ) Stück . 
in den Einmonats-Libor-Futures gingen 614 ( 1959 ) Kontrakte um , in den Dreimonats-Libor-Futures 2021 ( 1435 ) Stück . 
auf den Bund-Future wurden 969 ( 615 ) Call-und 545 ( 300 ) Put-Optionen gehandelt , auf den Bobl-Future 600 ( 3300 ) Calls und 1950 ( 1010 ) Puts . 
auf den US-Dollar gingen 1098 ( 1666 ) Calls und 580 ( 1007 ) Puts um .
