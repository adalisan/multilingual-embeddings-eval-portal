Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Montag deutlich gefallen . 
nach Angaben der DTB wurden bis 16.00 Uhr 26.186 ( Freitag 57.828 ) Kontrakte umgesetzt , davon 13.112 ( 36.289 ) Calls und 13.074 ( 21.539 ) Puts . 
die Call-Put-Ratio fiel auf 1,00 ( 1,68 ) . 
Umsatzspitzenreiter waren Siemens mit 3088 Calls und 3537 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 30.000 ( 108.581 ) Calls und 36.064 ( 62.054 ) Puts . 
der September-Dax-Future notierte um 16 Uhr nach einem Umsatz von 14.538 ( 17.622 ) Kontrakten mit 3775,5 ( 3800 ) Punkten , der September-MDax-Future nach einem Umsatz von 280 ( 276 ) Kontrakten mit 3960,5 ( 3985 ) Punkten . 
im September-Bund-Future wurden 80.376 ( 73.648 ) Kontrakte gehandelt , im Bobl-Future 38.005 ( 44.581 ) und im September-Schatz-Future 9050 ( 7547 ) Stück . 
in den Einmonats-Libor-Futures gingen 51 ( null ) Kontrakte um , in den Dreimonats-Libor-Futures 1130 ( 2163 ) Stück . 
auf den Bund-Future wurden 1000 ( 305 ) Call-und 571 ( 291 ) Put-Optionen gehandelt , auf den Bobl-Future 1410 ( 4380 ) Calls und 2633 ( 1730 ) Puts . 
auf den US-Dollar gingen 82 ( 507 ) Calls und 429 ( 528 ) Puts um .
