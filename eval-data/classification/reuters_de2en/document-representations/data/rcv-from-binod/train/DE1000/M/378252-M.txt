Frankfurt ( Reuter ) - An der DTB hat der Umsatz mit Aktienoptionen am Montag abgenommen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 50.943 ( Freitag 52.974 ) Kontrakte umgesetzt , davon 36.820 ( 34.095 ) Calls und 14.123 ( 18.879 ) Puts . 
die Call-Put-Ratio betrug 2,60 ( 1,80 ) . 
Umsatzspitzenreiter waren Siemens mit 6319 Calls und 1610 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 125.784 ( 68.422 ) Calls und 88.238 ( 55.683 ) Puts . 
der Dax-Volatilitäts-Index gab auf 15,86 ( 17,05 ) nach . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 13.074 ( 16.857 ) Kontrakten mit 3568,0 ( 3501,0 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 348 ( 514 ) Kontrakten mit 3640 ( 3595 ) Punkten . 
im Juni-Bund-Future wurden 27.239 ( 92.698 ) Kontrakte gehandelt , im Bobl-Future 10.199 ( 61.072 ) und im Juni-Schatz-Future 2696 ( 8601 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 200 ( 250 ) Kontrakte um , im Dreimonats-Libor-Future waren es 719 ( 2123 ) . 
auf den Bund-Future wurden 98 ( 1129 ) Call-und 536 ( 560 ) Put-Optionen gehandelt , auf den Bobl-Future 190 ( 3850 ) Calls und 602 ( 5600 ) Puts . 
auf den US-Dollar gingen 1101 ( 195 ) Call-und 140 ( 150 ) Put-Optionen um .
