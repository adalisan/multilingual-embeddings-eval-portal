Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Donnerstag bis 16.00 Uhr im Tagesvergleich gestiegen . 
bis zu diesem Zeitpunkt wurden 26.769 ( 32.194 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 13.228 ( 18.492 ) auf Calls und 13.541 ( 13.702 ) auf Puts . 
die Call-Put-Ratio fiel auf 0,97 ( 1,34 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 1974 Calls und 1612 Puts . 
auf den Dax wurden 30.769 ( 24.972 ) Calls und 47.041 ( 33.384 ) Puts gehandelt . 
der September-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 15.303 ( 13.263 ) Kontrakten mit 2584 ( 2563,5 ) Punkten . 
der Dax-Volatilitätsindex fiel auf 11,37 ( 11,47 ) . 
im Dezember-Bund-Future wurden bis 16.00 Uhr 48.580 ( 46.933 ) Kontrakte umgesetzt , im Dezember-Bobl-Future 67.799 ( 64.796 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 288 ( 26 ) Call-und 504 ( 403 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 1477 ( 1100 ) Calls und 1507 ( 100 ) Puts .
