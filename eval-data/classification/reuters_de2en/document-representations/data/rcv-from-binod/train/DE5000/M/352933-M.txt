Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Dienstag gegenüber Montag gesunken . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 34.551 ( 35.708 ) Kontrakte umgesetzt , davon 23.375 ( 18.788 ) Calls und 11.176 ( 16.920 ) Puts . 
die Call-Put-Ratio betrug 2,09 ( 1,11 ) . 
Umsatzspitzenreiter waren Thyssen mit 7130 Calls und 1389 Puts . 
auf den Dax wurden 70.917 ( 27.413 ) Calls und 55.843 ( 43.149 ) Puts gehandelt . 
der Dax-Volatilitätsindex fiel auf 16,74 ( 17,54 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 19.832 ( 13.773 ) Kontrakten mit 3241,0 ( 3181,5 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 479 ( 361 ) Kontrakten bei 3336 ( 3307 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 54.081 ( 41.660 ) Kontrakte umgesetzt , im März-Bobl-Future 76.354 ( 37.534 ) . 
im Einmonats-Libor-Future gingen über alle Sichten 2400 ( 2087 ) Kontrakte um , im Dreimonats-Libor-Future waren es 5049 ( 3124 ) . 
auf den Bund-Future wurden keine ( 476 ) Call-und 609 ( 1510 ) Put-Optionen gehandelt , auf den Bobl-Future 776 ( 800 ) Calls und 1562 ( 1500 ) Puts . 
auf den US-Dollar wurden bis 16.00 Uhr 970 ( 948 ) Call-und 182 ( 1158 ) Put-Optionen gehandelt .
