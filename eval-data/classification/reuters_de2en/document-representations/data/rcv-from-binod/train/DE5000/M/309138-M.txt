Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Donnerstag bis 16.00 Uhr gegenüber dem Vortag gestiegen . 
bis zu diesem Zeitpunkt wurden 39.822 ( 32.659 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 25.822 ( 22.437 ) auf Calls und 14.000 ( 10.222 ) auf Puts . 
die Call-Put-Ratio fiel auf 1,84 ( 2,19 ) . 
Umsatzspitzenreiter waren erneut Daimler mit 5301 ( 5954 ) Calls und 2640 ( 1373 ) Puts . 
auf den Dax wurden 39.635 ( 37.055 ) Calls und 38.710 ( 31.527 ) Puts gehandelt . 
der Dezember-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 11.493 ( 14.137 ) Kontrakten mit 2733 ( 2722 ) Punkten . 
der Dezember-MDax-Future lag nach einem Umsatz von 239 ( 827 ) Kontrakten bei 3013 ( 3030 ) Punkten . 
der Dax-Volatilitätsindex fiel auf 10,84 ( 10,90 ) . 
im Dezember-Bund-Future wurden bis 16.00 Uhr 78.172 ( 65.898 ) Kontrakte umgesetzt , im Dezember-Bobl-Future 58.101 ( 63.221 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 87 ( 200 ) Call-und 105 ( 12 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 1370 ( 900 ) Calls und 920 ( 4800 ) Puts .
