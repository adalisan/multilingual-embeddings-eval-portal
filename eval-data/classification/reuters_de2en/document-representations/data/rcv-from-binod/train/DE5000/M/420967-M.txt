Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Mittwoch bis 16.00 Uhr gegenüber dem Vortag gestiegen . 
bis zu diesem Zeitpunkt wurden 24.297 ( Dienstag 20.873 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 15.005 ( 11.961 ) auf Calls und 9292 ( 8912 ) auf Puts . 
die Call-Put-Ratio stieg auf 1,61 ( 1,34 ) . 
Umsatzspitzenreiter waren erneut Deutsche Bank mit 2771 ( 3103 ) Calls und 1299 ( 1719 ) Puts . 
auf den Dax wurden 33.775 ( 24.257 ) Calls und 59.080 ( 40.375 ) Puts gehandelt . 
der Dezember-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 12.841 ( 12.646 ) Kontrakten mit 2713,0 ( 2717,5 ) Punkten . 
der Dezember-MDax-Future lag nach einem Umsatz von 96 ( 226 ) Kontrakten bei unverändert 3031 Punkten . 
der Dax-Volatilitätsindex stieg auf 10,45 ( 10,32 ) . 
im Dezember-Bund-Future wurden bis 16.00 Uhr 74.560 ( 62.668 ) Kontrakte umgesetzt , im Dezember-Bobl-Future 47.956 ( 68.093 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 302 ( 610 ) Call-und 690 ( 1695 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 870 ( 1335 ) Calls und 4280 ( 2500 ) Puts .
