Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Mittwoch bis 16.00 Uhr im Tagesvergleich wieder gesunken . 
bis zu diesem Zeitpunkt wurden 31.186 ( 37.185 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 18.097 ( 19.816 ) auf Calls und 13.089 ( 17.369 ) auf Puts . 
die Call-Put-Ratio stieg auf 1,38 ( 1,14 ) . 
Umsatzspitzenreiter waren Daimler-Benz mit 4366 Calls und 1860 Puts . 
auf den Dax wurden 19.688 ( 27.481 ) Calls und 29.425 ( 30.171 ) Puts gehandelt . 
der September-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 9143 ( 11.209 ) Kontrakten bei 2548,5 ( 2563,5 ) Punkten . 
der Dax-Volatilitätsindex fiel auf 11,04 ( 11,07 ) . 
im September-Bund-Future wurden 41.175 ( 46.353 ) Kontrakte umgesetzt , im Bobl-Future 84.613 ( 63.043 ) Stück . 
im Fibor-Future gingen wieder keine Kontrakte um . 
auf den Bund-Future wurden 112 ( 35 ) Call-und 100 ( 100 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 650 ( 2538 ) Calls und 1700 ( 300 ) Puts .
