Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) hat am Mittwoch bis 16.00 Uhr im Tagesvergleich weiter zugelegt . 
bis zu diesem Zeitpunkt wurden 37.185 ( 23.542 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 19.816 ( 13.456 ) auf Calls und 17.369 ( 10.086 ) auf Puts . 
die Call-Put-Ratio sank auf 1,14 ( 1,33 ) . 
auf den Dax wurden bis 16 Uhr 27.481 ( 35.799 ) Calls und 30.171 ( 40.978 ) Puts gehandelt . 
der September-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 11.209 ( 11.876 ) Kontrakten bei 2563,5 ( 2570,0 ) Punkten . 
der Dax-Volatilitätsindex fiel auf 11,07 ( 11,20 ) . 
im September-Bund-Future wurden bis 16 Uhr 46.353 ( 55.118 ) Kontrakte umgesetzt , im Bobl-Future 63.043 ( 89.950 ) Stück . 
im Fibor-Future gingen wieder keine Kontrakte um . 
auf den Bund-Future wurden 35 ( 210 ) Call-und 100 ( 438 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 2538 ( 1198 ) Calls und 300 ( 250 ) Puts .
