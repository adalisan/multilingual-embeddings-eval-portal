Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) hat am Mittwoch bis 16.00 Uhr gegenüber dem Vortag zugenommen . 
bis zu diesem Zeitpunkt wurden 33.256 ( 29.753 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 19.443 ( 15.760 ) auf Calls und 13.813 ( 13.993 ) auf Puts . 
die Call-Put-Ratio stieg auf 1,40 ( 1,12 ) . 
Umsatzspitzenreiter waren Daimler mit 3168 Calls und 1047 Puts . 
auf den Dax wurden 67.277 ( 20.156 ) Calls und 86.565 ( 34.111 ) Puts gehandelt . 
der Dezember-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 24.190 ( 13.386 ) Kontrakten mit 2695,0 ( 2732,0 ) Punkten . 
der Dezember-MDax-Future lag nach einem Umsatz von 427 ( 534 ) Kontrakten bei 2966,0 ( 2988,0 ) Punkten . 
der Dax-Volatilitätsindex stieg auf 11,19 ( 10,70 ) . 
im Dezember-Bund-Future wurden bis 16.00 Uhr 71.180 ( 52.120 ) Kontrakte umgesetzt , im Dezember-Bobl-Future 70.925 ( 50.424 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 648 ( 92 ) Call-und 264 ( 178 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 2815 ( 4630 ) Calls und 1620 ( 1403 ) Puts .
