Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Donnerstag gegenüber dem Vortag gesunken . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 34.234 ( 39.078 ) Stück umgesetzt , davon 22.443 ( 28.123 ) Calls und 11.791 ( 10.955 ) Puts . 
die Call-Put-Ratio betrug 1,90 ( 2,56 ) . 
Umsatzspitzenreiter waren erneut Siemens mit 5569 ( 5169 ) Calls und 3295 ( 2045 ) Puts . 
auf den Dax wurden 39.158 ( 71.357 ) Calls und 67.934 ( 72.491 ) Puts gehandelt . 
der Dax-Volatilitätsindex fiel auf 15,03 ( 15,22 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 18.037 ( 19.576 ) Kontrakten mit 3121 ( 3123 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 357 ( 744 ) Kontrakten bei 3259 ( 3264 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 86.560 ( 86.608 ) Kontrakte umgesetzt , im März-Bobl-Future 88.349 ( 59.988 ) . 
im Einmonats-Libor-Future gingen über alle Sichten 3228 ( 1686 ) Kontrakte um , im Dreimonats-Libor-Future waren es 12.872 ( 2969 ) .
