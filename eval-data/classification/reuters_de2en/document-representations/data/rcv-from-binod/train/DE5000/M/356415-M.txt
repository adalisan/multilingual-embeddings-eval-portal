Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Mittwoch gegenüber Dienstag wenig verändert geblieben . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 50.472 ( 52.791 ) Kontrakte umgesetzt , davon 31.319 ( 36.394 ) Calls und 19.153 ( 16.307 ) Puts . 
die Call-Put-Ratio betrug 1,63 ( 2,21 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 4486 Calls und 1973 Puts . 
auf den Dax wurden 74.947 ( 68.076 ) Calls und 92.215 ( 75.873 ) Puts gehandelt . 
der Dax-Volatilitätsindex stieg auf 17,56 ( 17,34 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 30.487 ( 27.780 ) Kontrakten mit 3387,5 ( 3335,0 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 319 ( 1102 ) Kontrakten bei 3476,0 ( 3458,0 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 27.079 ( 53.527 ) Kontrakte umgesetzt , im März-Bobl-Future 39.085 ( 76.650 ) . 
im Einmonats-Libor-Future gingen über alle Sichten 638 ( 942 ) Kontrakte um , im Dreimonats-Libor-Future waren es 5462 ( 4098 ) . 
auf den Bund-Future wurden 280 ( 914 ) Call-und 297 ( 1970 ) Put-Optionen gehandelt , auf den Bobl-Future 2100 ( 500 ) Calls und 202 ( 2101 ) Puts . 
auf den US-Dollar wurden bis 16.00 Uhr 281 ( 1220 ) Call-und 108 ( 795 ) Put-Optionen gehandelt .
