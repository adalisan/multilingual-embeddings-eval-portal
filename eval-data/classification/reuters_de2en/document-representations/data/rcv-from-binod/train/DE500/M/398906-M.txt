Frankfurt ( Reuter ) - An der DTB hat der Umsatz mit Aktienoptionen am Mittwoch weiter zugenommen . 
nach Angaben der DTB wurden bis 16.00 Uhr 43.861 ( Dienstag 28.230 ) Kontrakte umgesetzt , davon 31.113 ( 18.993 ) Calls und 12.748 ( 9237 ) Puts . 
die Call-Put-Ratio stieg auf 2,44 ( 2,05 ) . 
Umsatzspitzenreiter waren Hoechst mit 6308 Calls und 1378 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 43.110 ( 29.452 ) Calls und 75.326 ( 35.117 ) Puts . 
der September-Dax-Future notierte um 16 Uhr nach einem Umsatz von 14.240 ( 12.435 ) Kontrakten mit 3887,5 ( 3833,0 ) Punkten , der September-MDax-Future nach einem Umsatz von 319 ( 403 ) Kontrakten mit 3985 ( 3962 ) Punkten . 
im September-Bund-Future wurden 117.226 ( 66.224 ) Kontrakte gehandelt , im Bobl-Future 59.307 ( 46.338 ) und im September-Schatz-Future 9500 ( 5970 ) Stück . 
in den Einmonats-Libor-Futures gingen 103 ( keine ) Kontrakte um , in den Dreimonats-Libor-Futures 2388 ( 610 ) Stück . 
auf den Bund-Future wurden 444 ( eine ) Call-und 1606 ( 491 ) Put-Optionen gehandelt , auf den Bobl-Future 1325 ( 2225 ) Calls und 2345 ( 3424 ) Puts . 
auf den US-Dollar gingen 749 ( 442 ) Calls und 138 ( 823 ) Puts um .
