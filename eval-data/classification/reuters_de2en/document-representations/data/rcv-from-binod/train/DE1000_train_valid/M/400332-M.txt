Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Montag gefallen . 
nach Angaben der DTB wurden bis 16.00 Uhr 21.577 ( 25.479 ) Kontrakte umgesetzt , davon 16.038 ( 16.788 ) Calls und 5539 ( 8691 ) Puts . 
die Call-Put-Ratio stieg auf 2,89 ( 1,93 ) . 
Umsatzspitzenreiter waren Veba mit 2607 Calls und 303 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 85.525 ( 44.459 ) Calls und 70.925 ( 95.545 ) Puts . 
der September-Dax-Future notierte um 16 Uhr nach einem Umsatz von 16.781 ( 12.381 ) Kontrakten mit 4012 ( 3966,5 ) Punkten , der September-MDax-Future nach einem Umsatz von 561 ( 729 ) Kontrakten mit 4056 ( 4025 ) Punkten . 
im September-Bund-Future wurden 86.305 ( 58.048 ) Kontrakte gehandelt , im Bobl-Future 46.693 ( 41.325 ) und im September-Schatz-Future 6971 ( 5418 ) Stück . 
in den Einmonats-Libor-Futures gingen erneut keine Kontrakte um , in den Dreimonats-Libor-Futures 2800 ( 670 ) Stück . 
auf den Bund-Future wurden 596 ( 12 ) Call-und 1183 ( 1246 ) Put-Optionen gehandelt , auf den Bobl-Future 450 ( 1800 ) Calls und 520 ( 3175 ) Puts . 
auf den US-Dollar gingen 102 ( 173 ) Calls und ein ( ein ) Put um .
