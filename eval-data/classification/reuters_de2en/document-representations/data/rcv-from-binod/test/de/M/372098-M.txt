Frankfurt ( Reuter ) - An der DTB hat der Umsatz mit Aktienoptionen am Mittwoch abgenommen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 33.700 ( 50.115 ) Kontrakte umgesetzt , davon 22.449 ( 30.867 ) Calls und 11.251 ( 19.248 ) Puts . 
die Call-Put-Ratio betrug 1,99 ( 1,60 ) . 
Umsatzspitzenreiter waren erneut Daimler mit 5180 ( 5768 ) Calls und 2403 ( 3485 ) Puts . 
auf den Dax wurden 47.376 ( 57.016 ) Calls und 53.190 ( 60.694 ) Puts gehandelt . 
der Dax-Volatilitätsindex fiel auf 20,93 ( 22,55 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 16.779 ( 22.808 ) Kontrakten mit 3348 ( 3373,5 ) , der Juni-MDax-Future nach einem Umsatz von 319 ( 392 ) Kontrakten mit 3576 ( 3576 ) Punkten . 
im Juni-Bund-Future wurden 112.563 ( 106.771 ) Kontrakte gehandelt , im Bobl-Future 57.071 ( 78.269 ) und im Juni-Schatz-Future 6457 ( 8960 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten null ( 481 ) Kontrakte um , im Dreimonats-Libor-Future waren es 3216 ( 2325 ) . 
auf den Bund-Future wurden 673 ( 1122 ) Call-und 1385 ( 1891 ) Put-Optionen gehandelt , auf den Bobl-Future 4245 ( 215 ) Calls und 6614 ( 1235 ) Puts . 
auf den US-Dollar gingen 878 ( 838 ) Call-und 156 ( 457 ) Put-Optionen um .
