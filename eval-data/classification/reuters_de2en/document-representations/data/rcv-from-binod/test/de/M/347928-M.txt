Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Dienstag bis 16.00 Uhr im Tagesvergleich gestiegen . 
bis zu diesem Zeitpunkt wurden 24.162 ( Montag 16.392 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 12.750 ( 10.196 ) auf Calls und 11.412 ( 6196 ) auf Puts . 
die Call-Put-Ratio fiel auf 1,11 ( 1,64 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 2203 Calls und 1446 Puts . 
auf den Dax wurden 52.824 ( 25.230 ) Calls und 80.815 ( 24.512 ) Puts gehandelt . 
der September-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 22.122 ( 5299 ) Kontrakten mit 2519,5 ( 2544,0 ) Punkten . 
der Dax-Volatilitätsindex stieg weiter 13,22 ( 12,36 ) . 
im September-Bund-Future wurden bis 16.00 Uhr 44.885 ( 28.160 ) Kontrakte umgesetzt , im September-Bobl-Future 72.929 ( 46.525 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 651 ( 236 ) Call-und 204 ( 149 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 2920 ( 2200 ) Calls und 191 ( 1500 ) Puts .
