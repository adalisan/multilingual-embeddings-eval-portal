Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Donnerstag zurückgegangen . 
nach Angaben der DTB wurden bis 16.00 Uhr 28.096 ( 39.243 ) Kontrakte umgesetzt , davon 17.952 ( 24.819 ) Calls und 10.144 ( 14.424 ) Puts . 
die Call-Put-Ratio stieg auf 1,76 ( 1,72 ) . 
Umsatzspitzenreiter waren erneut Deutsche Bank mit 3046 ( 2975 ) Calls und 1401 ( 2411 ) Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 51.299 ( 72.303 ) Calls und 74.650 ( 91.278 ) Puts . 
der September-Dax-Future notierte um 16 Uhr nach einem Umsatz von 22.889 ( 17.835 ) Kontrakten mit 4030,0 ( 4086,5 ) Punkten , der September-MDax-Future nach einem Umsatz von 402 ( 1041 ) Kontrakten mit 4067 ( 4112 ) Punkten . 
im September-Bund-Future wurden 123.644 ( 136.032 ) Kontrakte gehandelt , im Bobl-Future 74.933 ( 77.455 ) und im September-Schatz-Future 10.850 ( 8653 ) Stück . 
in den Einmonats-Libor-Futures gingen 155 ( 100 ) Kontrakte um , in den Dreimonats-Libor-Futures 4415 ( 2335 ) Stück . 
auf den Bund-Future wurden 861 ( 544 ) Call-und 303 ( 1231 ) Put-Optionen gehandelt , auf den Bobl-Future 519 ( 2355 ) Calls und 1610 ( 4019 ) ( Puts . 
auf den Dollar gingen 802 ( 1078 ) Calls und 391 ( 786 ) Puts um .
