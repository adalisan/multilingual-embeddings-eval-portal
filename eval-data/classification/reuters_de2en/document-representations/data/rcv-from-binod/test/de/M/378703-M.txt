Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Dienstag gestiegen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 54.064 ( Montag 50.943 ) Kontrakte umgesetzt , davon 35.578 ( 36.820 ) Calls und 18.486 ( 14.123 ) Puts . 
die Call-Put-Ratio betrug 1,92 ( 2,60 ) . 
Umsatzspitzenreiter waren Veba mit 5531 Calls und 5417 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 77.420 ( 125.784 ) Calls und 83.096 ( 88.238 ) Puts . 
der Dax-Volatilitäts-Index stieg auf 16,79 ( 15,86 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 23.464 ( 13.074 ) Kontrakten mit 3566,5 ( 3568,0 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 394 ( 348 ) Kontrakten mit 3625 ( 3640 ) Punkten . 
im Juni-Bund-Future wurden 74.186 ( 27.239 ) Kontrakte gehandelt , im Bobl-Future 56.314 ( 10.199 ) und im Juni-Schatz-Future 7943 ( 2696 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 350 ( 200 ) Kontrakte um , im Dreimonats-Libor-Future waren es 2957 ( 719 ) . 
auf den Bund-Future wurden 58 ( 98 ) Call-und 400 ( 536 ) Put-Optionen gehandelt , auf den Bobl-Future 1200 ( 190 ) Calls und 950 ( 602 ) Puts . 
auf den US-Dollar gingen 313 ( 1101 ) Calls und 430 ( 140 ) Puts um .
