Frankfurt ( Reuter ) - An der DTB hat der Umsatz mit Aktienoptionen am Dienstag zugenommen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 50.115 ( 38.218 ) Kontrakte umgesetzt , davon 30.867 ( 22.668 ) Calls und 19.248 ( 15.550 ) Puts . 
die Call-Put-Ratio betrug 1,60 ( 1,45 ) . 
Umsatzspitzenreiter waren Daimler mit 5768 Calls und 3485 Puts . 
auf den Dax wurden 57.016 ( 37.856 ) Calls und 60.694 ( 60.472 ) Puts gehandelt . 
der Dax-Volatilitätsindex stieg auf 22,55 ( 21,54 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 22.808 ( 18.029 ) Kontrakten mit 3373,5 ( 3299,5 ) , der Juni-MDax-Future nach einem Umsatz von 392 ( 234 ) Kontrakten mit 3576 ( 3541 ) Punkten . 
im Juni-Bund-Future wurden 106.771 ( 63.196 ) Kontrakte gehandelt , im Bobl-Future 78.269 ( 32.865 ) und im Juni-Schatz-Future 8960 ( 4657 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 481 ( 293 ) Kontrakte um , im Dreimonats-Libor-Future waren es 2325 ( 1144 ) . 
auf den Bund-Future wurden 1122 ( 710 ) Call-und 1891 ( 440 ) Put-Optionen gehandelt , auf den Bobl-Future 215 ( 4000 ) Calls und 1235 ( 933 ) Puts . 
auf den US-Dollar gingen 838 ( 781 ) Call-und 457 ( 10 ) Put-Optionen um .
