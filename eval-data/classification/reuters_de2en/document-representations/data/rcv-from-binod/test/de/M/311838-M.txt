Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Donnerstag bis 16.00 Uhr gegenüber dem Vortag geschrumpft . 
bis zu diesem Zeitpunkt wurden 30.071 ( 33.256 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 16.729 ( 19.443 ) auf Calls und 13.342 ( 13.813 ) auf Puts . 
die Call-Put-Ratio fiel auf 1,25 ( 1,40 ) . 
Umsatzspitzenreiter waren Siemens mit 2088 Calls und 2334 Puts . 
auf den Dax wurden 32.894 ( 67.277 ) Calls und 46.897 ( 86.565 ) Puts gehandelt . 
der Dezember-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 17.473 ( 24.190 ) Kontrakten mit 2695,5 ( 2695,0 ) Punkten . 
der Dezember-MDax-Future lag nach einem Umsatz von 293 ( 427 ) Kontrakten bei 2958,0 ( 2966,0 ) Punkten . 
der Dax-Volatilitätsindex stieg auf 11,31 ( 11,19 ) . 
im Dezember-Bund-Future wurden bis 16.00 Uhr 60.660 ( 71.180 ) Kontrakte umgesetzt , im Dezember-Bobl-Future 81.399 ( 70.925 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 97 ( 688 ) Call-und 326 ( 264 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 2350 ( 2815 ) Calls und 3173 ( 1620 ) Puts .
