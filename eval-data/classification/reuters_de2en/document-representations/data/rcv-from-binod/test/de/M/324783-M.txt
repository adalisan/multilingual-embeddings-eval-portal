Frankfurt ( Reuter ) - Die starken Schwingungen an den deutschen Finanzmärkten haben der DTB am Freitag einen Rekordumsatz beschert . 
wie die DTB mitteilte , wurde bereits bis 16.00 Uhr mit einem Umsatz von rund 742.000 Kontrakten in allen Produkten der bisherige Tagesrekord vom 2. März 1994 ( rund 621.000 Kontrakte ) übertroffen . 
bei den Aktienoptionen fiel der Umsatz bis 16.00 Uhr im Vergleich zum Vortag leicht auf 36.645 ( 37.343 ) Stück . 
auf den Dax wurden bis 16.00 Uhr 129.359 ( 103.925 ) Calls und 206.306 ( 90.835 ) Puts gehandelt . 
der Dezember-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 47.534 ( 24.794 ) Kontrakten mit 2819,5 ( 2896,0 ) Punkten . 
der Dezember-MDax-Future lag nach einem Umsatz von 611 ( 199 ) Kontrakten bei 2860 ( 2925 ) Punkten . 
der Dax-Volatilitätsindex stieg auf 16,90 ( 13,60 ) . 
im März-Bund-Future wurden bis 16.00 Uhr 161.150 ( 91.064 ) Kontrakte umgesetzt , im März-Bobl-Future 124.419 ( 90.998 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 2566 ( 4950 ) Kontrakte um . 
auf den Bund-Future wurden 655 ( 113 ) Call-und 730 ( 194 ) Put-Optionen gehandelt , auf den Bobl-Future 458 ( 1150 ) Calls und 744 ( 902 ) Puts .
