Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Dienstag bis 16.00 Uhr im Tagesvergleich gestiegen . 
bis zu diesem Zeitpunkt wurden 31.838 ( Montag : 20.015 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 19.807 ( 11.878 ) auf Calls und 12.031 ( 8137 ) auf Puts . 
die Call-Put-Ratio stieg auf 1,64 ( 1,45 ) . 
Umsatzspitzenreiter waren erneut Deutsche Bank mit 2372 ( 3155 ) Calls und 1901 ( 1950 ) Puts . 
auf den Dax wurden 49.726 ( 37.481 ) Calls und 47.394 ( 49.221 ) Puts gehandelt . 
der September-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 15.749 ( 12.347 ) Kontrakten mit 2568,5 ( 2554 ) Punkten . 
der Dax-Volatilitätsindex fiel auf 11,18 ( 11,72 ) . 
im Dezember-Bund-Future wurden bis 16.00 Uhr 43.810 ( 31.038 ) Kontrakte umgesetzt , im Dezember-Bobl-Future 51.370 ( 40.382 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 67 ( 35 ) Call-und 641 ( 411 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 1941 ( 960 ) Calls und 3650 ( 7128 ) Puts .
