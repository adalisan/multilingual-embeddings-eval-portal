Frankfurt ( Reuter ) - An der DTB hat der Umsatz mit Aktienoptionen am Freitag zugenommen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 52.974 ( 34.014 ) Kontrakte umgesetzt , davon 34.095 ( 24.439 ) Calls und 18.879 ( 9575 ) Puts . 
die Call-Put-Ratio betrug 1,80 ( 2,55 ) . 
der Umsatz mit Dax-Optionen verteilte sich auf 68.422 ( 41.546 ) Calls und 55.683 ( 53.638 ) Puts . 
der Dax-Volatilitäts-Index gab auf 17,05 ( 18,05 ) nach . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 16.857 ( 16.821 ) Kontrakten mit 3501,0 ( 3436,0 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 514 ( 168 ) Kontrakten mit 3595 ( 3585 ) Punkten . 
im Juni-Bund-Future wurden 92.698 ( 89.443 ) Kontrakte gehandelt , im Bobl-Future 61.072 ( 43.900 ) und im Juni-Schatz-Future 8601 ( 7215 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 250 ( 1111 ) Kontrakte um , im Dreimonats-Libor-Future waren es 2123 ( 2114 ) . 
auf den Bund-Future wurden 1129 ( 898 ) Call-und 560 ( 1800 ) Put-Optionen gehandelt , auf den Bobl-Future 3850 ( 439 ) Calls und 5600 ( 300 ) Puts . 
auf den US-Dollar gingen 195 ( 66 ) Call-und 150 ( 170 ) Put-Optionen um .
