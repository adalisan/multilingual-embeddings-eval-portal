Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) hat am Dienstag bis 16.00 Uhr im Tagesvergleich wieder zugenommen . 
bis zu diesem Zeitpunkt wurden 23.542 ( 17.011 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 13.456 ( 8319 ) auf Calls und 10.086 ( 8692 ) auf Puts . 
die Call-Put-Ratio stieg auf 1,33 ( 0,95 ) . 
Umsatzspitzenreiter waren Daimler mit 1914 Calls und 1509 Puts . 
auf den Dax wurden bis 16 Uhr 35.799 ( 17.012 ) Calls und 40.978 ( 23.182 ) Puts gehandelt . 
der September-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 11.876 ( 6797 ) Kontrakten bei 2570,0 ( 2560,0 ) Punkten . 
der Dax-Volatilitätsindex fiel auf 11,20 ( 11,24 ) . 
im September-Bund-Future wurden bis 16 Uhr 55.118 ( 25.475 ) Kontrakte umgesetzt , im Bobl-Future 89.950 ( 20.798 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 210 ( 69 ) Call-und 438 ( 630 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 1198 ( 454 ) Calls und 250 ( 200 ) Puts .
