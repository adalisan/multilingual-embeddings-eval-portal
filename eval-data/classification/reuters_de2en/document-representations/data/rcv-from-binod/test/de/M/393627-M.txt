Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Montag weiter gesunken . 
nach Angaben der DTB wurden bis 16.00 Uhr 21.944 ( Freitag 28.296 ) Kontrakte umgesetzt , davon 13.789 ( 16.478 ) Calls und 8155 ( 7818 ) Puts . 
die Call-Put-Ratio betrug 1,69 ( 2,10 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 1868 Calls und 1404 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 44.209 ( 75.295 ) Calls und 44.586 ( 65.559 ) Puts . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 14.609 ( 18.753 ) Kontrakten mit 3753,5 ( 3750,5 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 884 ( 788 ) Kontrakten mit 3925,0 ( 3928 ) Punkten . 
im September-Bund-Future wurden 78.047 ( 98.709 ) Kontrakte gehandelt , im Bobl-Future 67.545 ( 70.643 ) und im September-Schatz-Future 10.132 ( 11.592 ) Stück . 
in den Einmonats-Libor-Futures gingen keine ( 50 ) Kontrakte um , in den Dreimonats-Libor-Futures 1227 ( 3581 ) Stück . 
auf den Bund-Future wurden 140 ( 255 ) Call-und 222 ( 5210 ) Put-Optionen gehandelt , auf den Bobl-Future 800 ( 1700 ) Calls und 1201 ( 1160 ) Puts . 
auf den US-Dollar gingen 393 ( 473 ) Calls und 176 ( 70 ) Puts um .
