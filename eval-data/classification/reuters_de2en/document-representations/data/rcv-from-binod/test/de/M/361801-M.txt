Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Mittwoch gefallen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 37.935 ( 40.264 ) Kontrakte umgesetzt , davon 23.796 ( 23.717 ) Calls und 14.139 ( 16.547 ) Puts . 
die Call-Put-Ratio betrug 1,68 ( 1,43 ) . 
Umsatzspitzenreiter waren Bayer mit 3118 Calls und 2818 Puts . 
auf den Dax wurden 36.988 ( 47.707 ) Calls und 37.408 ( 99.851 ) Puts gehandelt . 
der Dax-Volatilitätsindex stieg auf das neue Jahreshoch von 20,61 ( 20,53 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 39.488 ( 44.105 ) Kontrakten mit 3306,5 ( 3321,5 ) , der März-MDax-Future nach einem Umsatz von 1184 ( 713 ) Kontrakten bei 3475 ( 3477,5 ) Punkten . 
im Juni-Bund-Future wurden 91.566 ( 172.923 ) Kontrakte gehandelt , im Juni-Bobl-Future 83.606 ( 117.223 ) und im Juni-Schatz-Future 12.829 ( 29.234 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 475 ( 945 ) Kontrakte um , im Dreimonats-Libor-Future waren es 2896 ( 4123 ) . 
auf den Bund-Future wurden 670 ( 448 ) Call-und 731 ( 2453 ) Put-Optionen gehandelt , auf den Bobl-Future 1505 ( 8250 ) Calls und 1665 ( 6456 ) Puts . 
auf den US-Dollar wurden bis 16 Uhr 916 ( 1095 ) Call-und 253 ( 196 ) Put-Optionen umgesetzt .
