Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Donnerstag leicht gefallen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 27.682 ( 28.146 ) Kontrakte umgesetzt , davon 19.915 ( 19.594 ) Calls und 7767 ( 8552 ) Puts . 
die Call-Put-Ratio betrug 2,56 ( 2,29 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 2524 Calls und 1197 ( 1856 ) Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 33.035 ( 36.486 ) Calls und 82.031 ( 49.079 ) Puts . 
der Dax-Volatilitäts-Index fiel auf 18,70 ( 18,82 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 18.424 ( 19.232 ) Kontrakten mit 3676,5 ( 3659 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 343 ( 471 ) Kontrakten mit 3872 ( 3836 ) Punkten . 
im September-Bund-Future wurden 100.235 Kontrakte gehandelt , im Bobl-Future 75.934 und im September-Schatz-Future 8615 Stück . 
in den Einmonats-Libor-Futures gingen 363 ( 275 ) Kontrakte um , in den Dreimonats-Libor-Futures 1880 ( 2406 ) Stück . 
auf den Bund-Future wurden 68 ( 100 ) Call-und 190 ( 169 ) Put-Optionen gehandelt , auf den Bobl-Future 1950 ( 1620 ) Calls und 1700 ( 4559 ) Puts . 
auf den US-Dollar gingen 1625 ( 821 ) Calls und 1061 ( 783 ) Puts um .
