Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Mittwoch gegenüber dem Vortag gesunken . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 40.816 ( 48.034 ) Stück umgesetzt , davon 26.993 ( 34.452 ) Calls und 13.823 ( 13.582 ) Puts . 
die Call-Put-Ratio betrug 1.95 ( 2,53 ) . 
Umsatzspitzenreiter waren Siemens mit 4917 Calls und 4117 Puts . 
auf den Dax wurden 64.318 ( 70.931 ) Calls und 51.540 ( 88.374 ) Puts gehandelt . 
der Dax-Volatilitätsindex stieg auf 16,95 ( 16,20 ) . 
der März-Dax-Future notierte um 16 Uhr nach einem Umsatz von 22.125 ( 29.966 ) Kontrakten mit 3242,0 ( 3220,0 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 1043 ( 464 ) Kontrakten bei 3301 ( 3314 ) Punkten . 
im März-Bund-Future wurden bis 16.00 Uhr 72.525 ( 76.738 ) Kontrakte umgesetzt , im März-Bobl-Future 65.558 ( 72.231 ) . 
im Einmonats-Libor-Future gingen über alle Sichten 745 ( 1340 ) Kontrakte um , im Dreimonats-Libor-Future waren es 1833 ( 5070 ) . 
auf den Bund-Future wurden 189 ( 597 ) Call-und 607 ( 2047 ) Put-Optionen gehandelt , auf den Bobl-Future 6890 ( 3920 ) Calls und 3908 ( 4102 ) Puts . 
auf den US-Dollar wurden bis 16.00 Uhr 1110 ( 2028 ) Call-und 1819 ( 462 ) Put-Optionen umgesetzt .
