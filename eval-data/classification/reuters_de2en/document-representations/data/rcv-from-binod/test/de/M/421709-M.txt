Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der Deutschen Terminbörse ( DTB ) ist am Freitag bis 16.00 Uhr gegenüber dem Vortag gesunken . 
bis zu diesem Zeitpunkt wurden 18.216 ( Vortag 27.343 ) Kontrakte umgesetzt , wie die DTB in Frankfurt mitteilte . 
davon entfielen 10.947 ( 16.443 ) auf Calls und 7269 ( 10.900 ) auf Puts . 
die Call-Put-Ratio war mit 1,50 unverändert . 
Umsatzspitzenreiter waren Daimler mit 2485 Calls und 1032 Puts . 
auf den Dax wurden 34.180 ( 43.314 ) Calls und 32.661 ( 50.369 ) Puts gehandelt . 
der Dezember-Dax-Future notierte um 16.00 Uhr nach einem Umsatz von 14.083 ( 16.683 ) Kontrakten mit 2712,0 ( 2685,5 ) Punkten . 
der Dezember-MDax-Future lag nach einem Umsatz von 221 ( 122 ) Kontrakten bei 3015,0 ( 3012,0 ) Punkten . 
der Dax-Volatilitätsindex fiel auf 11,14 ( 11,18 ) . 
im Dezember-Bund-Future wurden bis 16.00 Uhr 71.105 ( 88.361 ) Kontrakte umgesetzt , im Dezember-Bobl-Future 50.117 ( 55.512 ) Stück . 
im Fibor-Future gingen erneut keine Kontrakte um . 
auf den Bund-Future wurden 280 ( 448 ) Call-und 280 ( 541 ) Put-Optionen gehandelt , auf den Bobl-Future waren es 350 ( 0 ) Calls und 1055 ( 155 ) Puts .
